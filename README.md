# Salary_Prediction
The goal of this project is to predict a person’s salary based on features such as:

Years of experience

Monthly Salary

Job rate

Department 

Annual Salary

Country

The Salary Prediction project is a machine learning application that predicts an individual’s salary based on various features such as department , country , job rate, location, and skills. The goal is to build a predictive model that helps organizations estimate fair compensation and assists individuals in understanding their earning potential.
